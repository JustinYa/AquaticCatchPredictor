Fishing Prediction System

Leveraging real-time weather data to enhance fishing success.

About the Project

This project uses machine learning algorithms to predict fishing success rates based on weather data such as temperature, pressure, and GPS location. The device can recommend optimal fishing spots, helping anglers maximize their chances of success.

Features

  • Real-time weather data integration using OpenWeather API
  • GPS-based location tracking
  • Machine learning predictions with SVM and K-means clustering
  • Offline mode for use in remote locations
  • Portable device powered by ESP32

Getting Started

To get started, you'll need an ESP32 microcontroller, a temperature and pressure sensor, and an internet connection for real-time data collection.

  1. Clone this repository
  2. Install dependencies using Arduino IDE
  3. Upload code to your ESP32 device
